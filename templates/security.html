Security

How we keep your data safe

🔒
No Data Storage
Form inputs are never written to any database or file. Each prediction is processed in memory and discarded immediately.
🛡️
Input Validation
All inputs are validated server-side before processing. Invalid or malicious inputs are rejected with clear error messages.
🔐
No Authentication Required
No accounts, passwords, or personal identifiers are required. You remain completely anonymous.
📦
Isolated ML Model
The ML model runs in an isolated Python environment. It only receives numerical features — no raw personal text is passed to the model.
🌐
Local Deployment
This app runs locally on your machine. No data leaves your network unless you deploy it to a public server.
⚠️
Educational Use
This is an academic project. For production financial applications, additional security measures (HTTPS, auth, audit logs) would be required.
